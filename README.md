# 🎬 Полировщик клипов (Clip Polisher)

Десктоп-приложение под Windows: превращает горизонтальный клип стрима (файл **или ссылку
на Twitch-клип**) в вертикальный ролик **9:16** под TikTok / YouTube Shorts / Reels —
с раскладкой (вебка + геймплей), анимированными субтитрами, брендингом и нормализованным
звуком. Всё локально, распознавание речи и кодирование — на вашей видеокарте.

---

## ⬇️ Скачать и установить

1. Открой раздел **[Releases](../../releases/latest)** и скачай **`ClipPolisher-Setup-*.exe`**.
2. Запусти установщик. Прав администратора **не требуется** — ставится в твою папку пользователя.
3. Windows может показать синее окно **«Windows защитила ваш компьютер»** (установщик без
   платной цифровой подписи — это нормально). Нажми **«Подробнее» → «Выполнить в любом случае»**.
4. Запусти «Полировщик клипов» из меню Пуск или с рабочего стола.

### Первый запуск — один раз докачает компоненты

Сам установщик компактный (~200 МБ). При **первом запуске** приложение докачает то, что
слишком тяжёлое для установщика (показывает прогресс):

- 🧠 **модель распознавания речи** Whisper large-v3 (~3 ГБ);
- 🎮 **GPU-библиотеки** (CUDA/cuDNN, ~2 ГБ) — только если у тебя видеокарта **NVIDIA**.

Всё кладётся в `%LOCALAPPDATA%\ClipPolisher\` и больше не качается. Нужен интернет **только
в первый раз**.

### Требования

- Windows 10/11 (64-бит).
- Для быстрой работы — **видеокарта NVIDIA** (распознавание на CUDA, кодирование на NVENC).
  Без NVIDIA приложение тоже работает, но распознавание идёт на процессоре — заметно медленнее.
- ~6 ГБ свободного места (приложение + докачанные модель и библиотеки).

---

## 🚀 Как пользоваться

1. **Вход** — вставь ссылку на Twitch-клип или выбери видеофайл, нажми «Загрузить».
2. **Раскладка** — выбери пресет и подвинь мышью зоны вебки/геймплея на холсте 9:16.
3. **Обрезка** — потяни ручки начала/конца прямо на дорожке со стоп-кадрами.
4. **Субтитры** — шрифт, анимация (Pop / Fade / Slide-up / Караоке), цензура мата (бип/тишина).
5. **Брендинг** — ник и значок площадки (Twitch/YouTube/Kick).
6. **Экспорт** — качество/FPS, папка и имя файла, кнопка **«Отрендерить клип»**.

Режим **«Пачкой»** — добавь несколько клипов и обработай их одним набором настроек.

Итоговый файл сохраняется туда и с тем именем, что ты указал. Если имя занято — рядом
создаётся `имя (2).mp4` (существующий не перезаписывается).

## 🛠️ Сборка из исходников (для разработчиков)

```powershell
# 1. Окружение
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
pip install pyinstaller                 # для сборки .exe
# FFmpeg (нужен и для запуска из исходников):
winget install Gyan.FFmpeg

# 2. Запуск из исходников
python app.py

# 3. Сборка .exe (onedir → dist\ClipPolisher\)
pyinstaller --noconfirm --clean clip-polisher.spec

# 4. Установщик (нужен Inno Setup: winget install JRSoftware.InnoSetup)
&"$env:LOCALAPPDATA\Programs\Inno Setup 6\ISCC.exe" installer\clip-polisher.iss
# → installer\Output\ClipPolisher-Setup-*.exe
```

CLI-прогон всего пайплайна без интерфейса:

```powershell
python -m core.pipeline "<клип или URL>" --preset A --anim pop --nick "ник" --platform twitch --out out\final.mp4
```

### Архитектура

- `core/` — вся логика обработки (ffmpeg, whisper, субтитры, рендер), **без Qt**.
- `ui/` — тонкая оболочка PySide6.
- `core/provision.py` — докачка модели и CUDA при первом запуске.
- `core` никогда не импортирует `ui`.

## Стек

Python 3.11+ · PySide6 · FFmpeg (h264_nvenc / libx264) · faster-whisper (large-v3, CUDA) ·
yt-dlp · субтитры .ass (libass).

## Лицензия

Шрифт PT Sans — ParaType (OFL, см. `assets/fonts/OFL.txt`). FFmpeg и модель Whisper —
под своими лицензиями (докачиваются отдельно).
